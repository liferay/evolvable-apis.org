Recipes (towards Evolvable APIs)

Recipes index

Making an API Evolvable is about how many different challenges are designed or implemented when we build it. Maybe you do not need to solve all of them for your case. In this section, we will provide some individual recipes on how to address each of this facets.

The following articles, will address specific needs and challenges that might appear while designing an API and suggests evolvable solutions. Or, maybe you do not need to make your API evolvable but you find that you can get the benefit of some of them.
